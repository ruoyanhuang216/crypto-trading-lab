### P2 — Volatility signals ✅

Added two `BaseSignal` subclasses to `signals/volatility/`:

#### `BBWidth(period=20, num_std=2.0)`

```python
from signals import BBWidth
out = BBWidth().compute(df)
# adds: bb_mid, bb_upper, bb_lower, bb_width
```

- `bb_width = (upper − lower) / mid` — dimensionless band squeeze ratio
- Compresses before breakouts; expands during high-volatility / trending periods
- Formula matches `ml/features/technical.py` exactly (same `rolling` + `std`)

#### `ATRVolatility(period=14)`

```python
from signals import ATRVolatility
out = ATRVolatility().compute(df)
# adds: atr (price units), atr_pct (dimensionless, primary)
```

- `atr_pct = ATR / close` — relative bar volatility, independent of price level
- Wilder smoothing (`ewm`, `alpha=1/period`, `adjust=False`) matches `ml/features/technical._atr()`
- Raw `atr` retained as secondary output for stop-loss / position sizing calculations

Both are exported from `signals/__init__.py` alongside the existing trend signals.

**Sample output (last 3 daily bars, BTC 2022–2025):**

| date | bb_width | atr_pct |
|---|---|---|
| 2024-12-30 | 0.1606 | 0.0421 |
| 2024-12-31 | 0.1658 | 0.0420 |
| 2025-01-01 | 0.1687 | 0.0404 |

### P-ML5 — Extended 6yr dataset for regime-specific models (F12) ✅

**Motivation (F10):** Bull model in `RegimeEnsemble` only got 128–136 training bars per
fold on 3yr data, producing negative IC (−0.044 to −0.138). Hypothesis: 6yr data (2019–2025)
provides enough bull bars across multiple bull cycles.

**Implementation:** `notebooks/p_ml5_extended_dataset.ipynb` — 7 sections, fully executed.

Key implementation detail — **cache-guard** for 2019–2021 data:
```python
df_raw = fetch_ohlcv(symbol=SYMBOL, timeframe="1d", since=SINCE, until=UNTIL)
if df_raw.index.min() > pd.Timestamp("2020-01-01", tz="UTC"):
    df_raw = fetch_ohlcv(..., use_cache=False)  # re-fetch full range from exchange
```

**Dataset:** 2,171 usable bars (2019-01-22 → 2024-12-31)

**Regime distribution (6yr):** bull=31.6%, bear=25.6%, ranging=42.8%
(vs P-ML4 3yr: bull=31.9%, bear=17.6%, ranging=50.5% — bear heavier in 2019–2022)
